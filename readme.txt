Weather App

 Overview
The Weather App provides real-time weather information and forecasts using the OpenWeather API. Users can retrieve current weather conditions, forecast data, and visualize weather patterns through dynamic charts. The app also features a chatbot for easy interaction.

 Features
- Current Weather: Fetch and display the current weather for a specified city.
- Weather Forecast: Get a 5-day weather forecast with detailed temperature and condition information.
- Dynamic Charts: Visualize weather data using bar, line, and doughnut charts.
- Chatbot Integration: Use natural language queries to ask about the weather (e.g., "What is the weather today?" or "What is the weather in [city]?").
- Sorting and Filtering: Sort forecast data by temperature and filter for rainy days.
- User-Friendly Interface: Easy-to-use input fields and buttons for interaction.

 Technologies Used
- HTML/CSS for structure and styling
- JavaScript for interactivity
- Chart.js for data visualization
- OpenWeather API for weather data
- Geolocation API for retrieving the user's location

 Usage
1. Enter a city name in the input field and click the "Get Weather" button to retrieve the current weather and forecast.
2. Use the sorting and filtering buttons to customize the displayed forecast data.
3. Interact with the chatbot by typing queries related to the weather in the chat box.
